# Julia Security Advisories

Welcome to the Julia Security Advisory Database documentation site.

This site provides a browsable, searchable interface to security advisories affecting packages in the Julia ecosystem.

## Overview

The Julia Security Advisory Database tracks security vulnerabilities in packages registered in the [Julia General registry](https://github.com/JuliaRegistries/General). Each advisory contains:

- **Advisory ID**: Unique identifier (JLSEC-YYYY-NNN format)
- **Affected Packages**: Package names and vulnerable version ranges
- **Aliases**: References to CVE, GHSA, and other vulnerability databases
- **Details**: Description, patches, workarounds, and references
